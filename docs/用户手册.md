# 用户手册

1. 初始化数据库：推荐双击 init_mysql_db.bat，脚本会自动运行 mysql/schema.sql 和 mysql/seed_data.sql。新建库会覆盖同名literature_management 数据库。
2. 启动平台：双击 start_mysql_app.bat，按提示输入 MySQL 用户名和密码；浏览器打开 http://127.0.0.1:8000。
3. 登录注册：普通用户可点击注册创建账号；管理员账号由 seed_data.sql 初始化。
4. 检索论文：在检索区输入标题、摘要、DOI、作者或机构；可填写年份或年份范围；可从关键词下拉点击添加多个关键词筛选。
5. 收藏论文与关键词：检索结果中点击收藏论文；收藏页上方可检索 keyword 表中的任意关键词并收藏；收藏关键词可作为筛选条件。
6. 上传论文：进入文献入库区，选择 PDF 上传解析，系统仅预填信息，用户确认后再保存。
7. 批量解析：在批量 PDF 解析区一次选择多篇 PDF，系统生成多个确认表单，检查后统一保存。
8. 添加作者：新增或修改论文时，作者必须从作者表选择；若列表中没有，可在添加作者模块输入姓名、邮箱、ORCID 后保存。ORCID 必填且唯一；同名但 ORCID 不同的作者可以同时存在。
9. 管理员审核：管理员可筛选 pending 论文并发布；active 可归档，archived 可重新发布；管理员也可修改或删除论文。
10. 导出数据：点击导出 JSON 或导出 CSV 下载当前论文数据。
11. 语言设置：在设置区切换中文/英文界面语言。
